# ECE 537 – Intro to Electrical Engineering

University of New Hampshire, ECE Department, Instructor: Dr. Sandeep Sunkavilli

# Course Overview

A course introducing fundamental electrical engineering concepts to non-ECE majors — circuit analysis, semiconductor devices, and analog/digital circuit design — with a strong hands-on lab component reinforcing every major topic.

# Textbook

*Electrical Engineering: Principles and Applications*, Hambley, 7th Edition.

## Structure

```
ECE537-Intro-to-Electrical-Engineering/
├── lectures/            # Weekly topic + lab index
├── labs/                # Weekly lab handouts and starter files
└── reference-designs/   # Instructor reference circuits and solved examples
```

> **Note:** Slide decks and full lecture materials are posted on Canvas, not in this public repo. The `lectures/` folder just tracks what topic was covered each week and which lab it pairs with.

## Hardware for the course

- **Electrical components** — breadboard, Resistors, Capacitors, Inductors, Diodes, BJTs, Op-Amps and TTL chips
- **Electrical sources and measuring devices** - AC/DC sources, Function generators and Oscilloscope

## Software for the course

- **Multisim** — breadboard-style circuit labs (DC analysis, BJT/MOSFET biasing)
- **Matlab**
