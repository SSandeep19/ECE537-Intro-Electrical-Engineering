# ECE 537 – Electrical Engineering Principles

Course repository for **ECE 537: Electrical Engineering Principles**, University of New Hampshire, ECE Department.

Textbook: *Electrical Engineering: Principles and Applications*, Hambley, 7th Edition.

## Structure

```
ECE537-Electrical-Engineering-Principles/
├── lectures/            # Weekly topic + lab index (no slides — see Canvas)
├── labs/                # Weekly lab handouts and starter files
└── reference-designs/   # Instructor reference circuits and solved examples
```

> **Note:** Slide decks and full lecture materials are posted on Canvas, not in this public repo. The `lectures/` folder just tracks what topic was covered each week and which lab it pairs with.

## Tools

- **Multisim** — breadboard-style circuit labs (DC analysis, BJT/MOSFET biasing)
- **LTspice** — op-amp, filter, and Bode plot / AC sweep labs
